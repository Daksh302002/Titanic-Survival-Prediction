# Titanic Survival Prediction

## Project Overview
Predict whether a passenger survived the Titanic disaster using machine learning.

## Dataset
- Source: Seaborn Titanic Dataset
- Rows: 891
- Target: survived

## Workflow
- Data Cleaning
- Missing Value Treatment
- Exploratory Data Analysis
- Feature Engineering
- One-Hot Encoding
- Model Training
- Hyperparameter Tuning
- Model Evaluation

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- Gaussian Naive Bayes
- XGBoost
- Gradient Boosting
- AdaBoost
